# ant-build

## Composer
Install the composer dependencies.
```
composer
```

## Build Workflow
```
composer
|-validate
|--compile
|---test
|----package
|-----verify
```

### Clean
Clean the target direcory.
```
clean
```

### phpdoc
Create the php doc.
```
phpdoc
```

## Tagging
Tag the source code
```
tagging
```

## Use
To use, create a main xml:

Copy the build.xml.dist as build.xml in project main.
